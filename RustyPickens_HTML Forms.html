Computer Literacy - Section 03

In Computer Literacy we learn the basic fundamentals of a computer and primary productivity software applications such as Word Processors, Spreadsheets, Presentation software, and basic security and privacy concepts.

Fundamentals of Cloud Computing

In Fundamentals of Cloud Computing we learn the basics of what cloud computing is, how to use it, and cover the major different areas of vendors and products offering the marketplace.

Internet Programming

In Internet Programming we learn the basics of how to program in HTML, CSS, and Javascript as well as learning how basic networking and web hosting works.
